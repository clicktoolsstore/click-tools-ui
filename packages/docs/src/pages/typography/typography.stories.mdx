## Typography Styles Usage

To provide flexibility in styling elements, we export a style class builder as

```jsx
typography({variant: <variantName>})
```

The class can be utilized in your <code>html/components</code> to achieve consistent typography styles.

## Basic Usage

For basic usage, apply the typography styles directly to your elements:

```jsx
import { typography } from "@click-tools-ui/react";
...
  // Apply typography styles using typography class in any html element
  <h1 className={typography({ varaint: 'h1' })} {...rest}>
    Sample
  </h1>;
```

## Advanced Usage with Wrappers

If you need to apply additional custom classes or wrap the element:

```jsx
import { FC, HTMLAttributes } from 'react'
import { typography, TypographyProps } from "@click-tools-ui/react";

type Props = HTMLAttributes<HTMLHeadingElement> & TypographyProps

export const Wrapper: FC<Props> = ({
  variant,
  className,
  children,
  ...rest
}) => {
  return (
    // typography({ variant, ['custom classes' , className] })
    <h2 className={typography({ variant, className })} {...rest}>
      {children}
    </h2>
  )
}
```
